# PareTerminal（パリタ＝ミナル）
異常検知を光ってお知らせ！工場でよくみるパトライトのシングナルタワーをreTerminalで制御できる拡張モジュールです。家でノイズキャンセリングイヤホンをしていても、宅配の訪問を見逃しません！


## システム構成
・LED積層信号灯 シグナル・タワー LRシリーズ LR4
・GROVEシステムの音センサー
・Raspberry Pi用高解像度オートフォーカスカメラモジュール（現在未搭載）
を搭載した拡張モジュールです。
